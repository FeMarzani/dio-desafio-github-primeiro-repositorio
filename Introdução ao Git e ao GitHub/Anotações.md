# Anotações úteis do módulo de introdução ao Git/GitHub

## COMANDOS NO TERMINAL GIT BASH
**mkdir** - Criação de pastas
**eccho > nome.txt** - Cria um arquivo .txt
**del nomepasta** - Deleta arquivo dentro da pasta nomeada
**rmdir nomepasta /S /Q** - Deleta pasta com tudo
**ctrl + l** - Limpa visualização do terminal
**cd nomepasta** - Muda o diretório
**ls** - Mostra que tipo de arquivos há na pasta
**ls -a** - Mostra se há arquivos ocultos
**git status** - Mostra se não há arquivos com necessidade de commit
**git add.** - Comando para adicionar arquivos novos ao repositório
**git commit -m "MENSAGEM"** - Comando para commitar
**Git push origin main** - Comando para lançar o commit no GitHub

## ANOTAÇÕES
O SHA-1 serve como uma função de Encriptação. Podendo ser considerada uma forma curta de representação de arquivo.
